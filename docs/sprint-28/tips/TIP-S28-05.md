## TASK

### 1) API routes

**Equipment:**

**`GET /api/production/equipment`** role authenticated
- Query: `?workCenterId=&status=`
- 200 `{ equipment: [...] }`.

**`POST /api/production/equipment`** role `production|admin`
- Body Zod: `{ code, name, workCenterId?, type?, model?, serialNumber? }`
- 201.

**`PUT /api/production/equipment/:id`** role `production|admin`
- Update field.

**`POST /api/production/equipment/:id/status`** role `production|admin`
- Body: `{ status: 'running'|'idle'|'maintenance'|'breakdown', notes? }`
- Update status.

**Downtime:**

**`GET /api/production/downtime`** role authenticated
- Query: `?workCenterId=&from=&to=&type=`
- 200 `{ records: [...] }`.

**`POST /api/production/downtime`** role `production|admin`
- Body Zod: `{ workCenterId, type: 'PLANNED'|'UNPLANNED'|'MAINTENANCE'|'BREAKDOWN'|'CHANGEOVER', reason, category?, workOrderId?, equipmentId? }`
- Behavior: tạo DowntimeRecord(startTime=now, endTime=null), reportedBy=user.
- Nếu equipmentId → update Equipment.status = 'maintenance' hoặc 'breakdown'.
- 201 `{ record }`.

**`POST /api/production/downtime/:id/resolve`** role `production|admin`
- Body: `{ resolution }`
- Set endTime=now, durationMinutes calc, resolvedBy=user, resolution.
- Nếu equipment liên quan → update Equipment.status = 'running'.

**`GET /api/production/maintenance-schedule?days=7`** role authenticated
- Trả Equipment có nextMaintenanceDate trong N ngày tới.
- 200 `{ upcoming: [...] }`.
